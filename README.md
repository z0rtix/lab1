# DynamicArray (C)
Реализация универсального динамического массива на языке C с поддержкой разных типов данных через механизм полиморфизма.

## Поддерживаемые типы
* `int`
* `double`
* `string`

Типы описываются через структуру `TypeInfo`, содержащую функции:
* копирования элемента
* сравнения
* печати
* освобождения памяти
Это позволяет контейнеру работать с различными типами без изменения кода.

# Основные операции
Контейнер поддерживает следующие операции:
* `push` — добавление элемента
* `pop` — удаление последнего элемента
* `get` — получение элемента по индексу
* `set` — изменение элемента
* `remove_at` — удаление по индексу
* `clear` — очистка массива

Работа с памятью:
* `reserve`
* `shrink_to_fit`

Алгоритмы:
* `sort`
* `map`
* `where`
* `copy`
* `concat`

# Тестирование
Для проверки работы контейнера реализованы тесты:
* тесты для `int`
* тесты для `string`
* тесты граничных случаев
* стресс-тест (10 млн операций)
Все тесты используют `assert`.

# Интерактивный режим
После запуска программы доступно меню, позволяющее:
* выбрать тип массива
* добавлять элементы
* удалять элементы
* сортировать
* применять `map`
* применять `where`
